# KARMA STELLA — カルマステラ

> 生年月日から読み解く、気質分析・方位・相性の占術ツール

## 概要

九星気学・西洋占星術・四柱推命の3占術エンジンを統合した、自己理解・意思決定支援ツールです。

- 🌟 **本命星・月命星・傾斜宮** — 九星気学による気質分析
- ♈ **西洋星座** — 12星座×属性による傾向分析
- 🧭 **吉方位** — 今月の吉凶方位（九星気学準拠）
- 💞 **相性診断** — 2名の生年月日から相性スコアを算出
- 🌙 **月齢連動** — リアルタイムの月相に基づくメッセージ

## 使い方

1. 生年月日を入力
2. 本命星・星座が自動算出される
3. 鑑定・詳細・相性の各タブで詳細を確認

## 技術仕様

- 純粋HTML/CSS/JavaScript（単一ファイル・外部依存なし）
- モバイルファースト設計
- 節入り日精密テーブル（1940〜2030年）内蔵
