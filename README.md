# KARMA STELLA — カルマステラ

> 星と魂の鑑定アプリ。九星気学 × 西洋占星術 × 四柱推命の三占術統合。

**URL:** https://karmastella.jp

---

## 概要

生年月日を入力するだけで、三つの占術エンジンがあなたの魂のタイプを分析します。  
108通りのソウルタイプ診断、詳細な性格分析、今日のカルマタスクを提供します。

---

## 機能

| 機能 | 内容 |
|---|---|
| ソウルタイプ診断 | 九星気学（9）× 星座（12）= 108パターンの性格分析 |
| 九星気学鑑定 | 本命星・月命星・傾斜宮の算出と解説 |
| 西洋占星術 | 星座・属性・カスプ判定 |
| 四柱推命 | 年柱・月柱・日柱・五行バランス |
| 吉方位 | 本命星に基づく吉方位・凶方位 |
| 相性診断 | 九星気学 × 星座の二軸相性スコア |
| カルマタスク | 星の配置に基づく今日の行動指針 |
| 魂の分析 | 6軸（本質・対人・仕事・意思決定・ストレス・成長）の詳細テキスト |

---

## 技術仕様

- **アーキテクチャ:** 単一HTMLファイル（外部依存なし・Google Fontsのみ）
- **ホスティング:** GitHub Pages（カスタムドメイン `karmastella.jp`）
- **占術エンジン:** 純粋JavaScript実装
  - 月柱：節入り日精密ルックアップテーブル（1940〜2030年対応）
  - 日柱：ユリウス通日方式 `(JDN + 49) % 60`
  - 本命星：`11 - digit_sum_of_birth_year`（業界標準）
- **データ:** `soul_descriptions.js`（108パターン × 6軸テキスト）を外部ファイルで管理
