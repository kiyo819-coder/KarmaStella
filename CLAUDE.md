# KARMA STELLA — プロジェクトガイド

## プロジェクト概要

占術（九星気学 × 四柱推命 × 星座）を統計的アプローチでユーザーに届けるWebアプリ。
「占いではなく、自己理解のためのツール」として、Instagram運用で認知を広め karmastella.jp に誘導する。

## リポジトリ構成

```
~/KarmaStella/              ← このリポジトリ（アプリ本体 + エージェントチーム統合）
├── index.html              ← Webアプリ本体
├── soul_descriptions.js    ← 108パターン魂データ
├── images/                 ← キャラクター画像
├── CNAME                   ← GitHub Pages カスタムドメイン
├── CLAUDE.md               ← このファイル
└── .claude/
    ├── agents/             ← サブエージェント定義
    │   ├── VISION.md       ← 全員共通の使命・誓い
    │   ├── TEAM_README.md  ← チーム設計図
    │   ├── sol.md          ← リーダー（統括・オーケストレーター）
    │   ├── vento.md        ← コンテンツ生成
    │   ├── aqua.md         ← トーンチェック
    │   └── ignis.md        ← 実行・投稿管理
    └── commands/
        └── post.md         ← 投稿一括コマンド（/post）
```

## エージェントチーム

| エージェント | 星座×九星 | 役割 | 起動タイミング |
|---|---|---|---|
| **SOL** 🌟 | 獅子座×五黄土星 | リーダー・オーケストレーター | 計画・意思決定・ワークフロー開始 |
| **VENTO** 🌀 | 双子座×三碧木星 | コンテンツ生成 | キャプション・ハッシュタグ生成 |
| **AQUA** 💧 | 蟹座×一白水星 | トーンチェック | 感情温度・NGワード確認 |
| **IGNIS** 🔥 | 牡羊座×九紫火星 | 実行・投稿管理 | チェックリスト・スケジュール管理 |

## Instagram投稿ワークフロー

```
ユーザー → SOL（指示）
  → VENTO（キャプション3案生成）
  → AQUA（トーンチェック・修正）
  → IGNIS（チェックリスト・投稿準備）
  → SOL（最終GO判定）→ ユーザーへ報告
```

**ショートカット：**
```
/post [芸能人名] [生年月日]
```

## 判断の最高基準

> 「これは、触れた人の魂を震わせるか？」
> YES → 進む　NO → 作り直す

## ターゲット

20〜35歳女性・スピリチュアル層・自己理解に興味がある人

## NGルール（全エージェント共通）

- 「運命」「宿命」の断定的使用
- 「必ず〜になります」という予言口調
- 「波動」「引き寄せ」の多用
- 断定口調（「あなたはこういう人です」）
- AQUAチェック前にIGNISへ渡すこと
